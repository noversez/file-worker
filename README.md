# File Prefix Cleaner

Desktop-приложение на Electron для массовой работы с файлами и папками по префиксам.

Функционал: ищет файлы и папки по префиксам, показывает предпросмотр, умеет переименовывать с учетом конфликтов имен и удалять в корзину.

## Запуск

1. Установите зависимости: `npm install`
2. Запустите приложение: `npm start`

## Сборка exe

1. Соберите portable-версию: `npm run dist`
2. Готовый файл появится в папке `dist` как `File Prefix Cleaner 2.0.0.exe`
3. Это обычный запускаемый файл для Windows, его можно переносить как один `.exe`

## Структура

- `main.js` — главный процесс Electron и файловая логика.
- `preload.js` — безопасный мост между окном и Node/Electron API.
- `renderer.js` — интерфейс и взаимодействие с пользователем.
- `index.html` и `styles.css` — разметка и оформление окна.
- `old-file-worker/` — архив старой PowerShell-версии.

## Что умеет

- Убирать префиксы из имен файлов и папок: `[QWERTY.COM] 1. Урок.mp4` -> `1. Урок.mp4`.
- Удалять файлы и папки, которые начинаются с одного из указанных префиксов.
- Принимать несколько префиксов: по одному в строке, через запятую или `;`.
- Искать только в выбранной папке или вместе с подпапками.
- Показывать предпросмотр перед действием.
- Удалять файлы и папки в корзину.
- Решать конфликты имен через `имя (2).ext`.
