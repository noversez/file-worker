# File Prefix Cleaner

**File Prefix Cleaner** — desktop-приложение для Windows, которое находит файлы и папки по заданным префиксам, показывает предпросмотр и затем переименовывает найденные элементы или переносит их в корзину.

Текущая версия переписана с PowerShell на Electron + TypeScript. Оригинальная PowerShell-версия сохранена в `powershell-version/`.

English version: [README-EN.md](README-EN.md)

## Возможности

- Поиск файлов и папок, имена которых начинаются с одного из указанных префиксов.
- Несколько префиксов: по одному в строке, через запятую или через `;`.
- Сканирование только выбранной папки или вместе с подпапками.
- Таблица предпросмотра перед любым действием.
- Выбор отдельных строк через чекбоксы.
- Переименование без перезаписи существующих файлов: при конфликте создаётся имя вида `name (2).ext`.
- Удаление в корзину, а не безвозвратное удаление.
- Экспорт предпросмотра в CSV.
- Кастомное окно подтверждения внутри приложения.
- Фирменная иконка приложения и `.exe`.

Пример переименования:

```text
[QWERTY.COM] Lesson 1.mp4 -> Lesson 1.mp4
```

## Как пользоваться

1. Выберите целевую папку.
2. Введите один или несколько префиксов.
3. При необходимости включите сканирование подпапок.
4. Нажмите «Сканировать для переименования» или «Сканировать для удаления».
5. Проверьте таблицу предпросмотра и снимите чекбоксы с элементов, которые не нужно обрабатывать.
6. Нажмите «Выполнить» и подтвердите действие в модальном окне.

## Установка для разработки

```bash
npm install
```

## Запуск

```bash
npm start
```

Команда сначала собирает TypeScript, затем запускает Electron.

## Сборка

Собрать portable `.exe` для Windows:

```bash
npm run dist
```

Готовый файл появится в папке `dist/`.

Собрать unpacked-версию быстрее:

```bash
npm run dist:dir
```

`npm run pack` оставлен как alias для unpacked-сборки.

## Скрипты

- `npm run build` — компиляция TypeScript в `build/`.
- `npm start` — сборка и запуск приложения.
- `npm run dist` — сборка portable `.exe`.
- `npm run dist:dir` — сборка unpacked-приложения.
- `npm run pack` — alias для `dist:dir`.

## Структура проекта

```text
assets/                 Иконки приложения: svg, png, ico
src/main/domain/        Чистые функции для работы с префиксами
src/main/application/   Сканирование, переименование, удаление в корзину
src/main/ipc/           IPC handlers, экспорт CSV, управление окном
src/main/window/        Создание BrowserWindow
src/preload/            Безопасный bridge между renderer и main process
src/renderer/           Логика интерфейса
src/shared/             Общие TypeScript-контракты
powershell-version/     Оригинальная PowerShell-реализация
```

## Безопасность

- Перед выполнением всегда проверяйте таблицу предпросмотра.
- Удаление переносит элементы в корзину.
- Переименование не перезаписывает существующие файлы и папки.
- Для рекурсивного режима приложение обрабатывает вложенные элементы от самых глубоких к верхним, чтобы корректно работать с папками.
